## Getting Started

To get a local copy up and running follow these simple steps.

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/YohannHommet/Perso_Suivis-Candidatures
```
2. Open VS Code
3. Composer install
3. Set your database environment
4. Launch migrations
5. Start server
6. Enjoy
